# AI-Job-replacement--Power-BI-Project
Built an interactive Power BI dashboard analyzing the impact of AI on jobs across industries. The project explores automation risk, salary changes, skill demand growth, and reskilling needs using DAX measures and advanced visuals to identify key drivers of workforce disruption.

# Dataset :

The dataset contains 15,000 job records with 20 variables, including:

Job Role
Industry
Country
Automation Risk (%)
AI Adoption Level
Salary Before and After AI
Skill Gap Index
Skill Demand Growth
Reskilling Urgency Score
AI Disruption Intensity

These variables help analyze how AI is transforming the workforce.

# Tools & Technologies Used :

Power BI
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization
